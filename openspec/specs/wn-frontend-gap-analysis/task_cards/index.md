# Task Cards: WriteNow 前端缺口补全

任务卡片按优先级分组：

- **P0**：核心可用性（设置入口、编辑器工具栏、状态栏基础、错误处理）
- **P1**：日常体验（文件管理、右键菜单、大纲导航、导出功能、帮助系统）
- **P2**：专业功能（可访问性、后端模块暴露第一批）
- **P3**：进阶功能（后端模块完整暴露、语义搜索 UI）

## 任务卡片规范

每张任务卡必须包含：

- Goal：任务目标
- Dependencies：前置依赖
- Expected File Changes：预期文件变更
- Acceptance Criteria：验收标准
- Tests：测试要求
