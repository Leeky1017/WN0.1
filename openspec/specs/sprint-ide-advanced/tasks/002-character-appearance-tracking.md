# 任务 002: 人物出场追踪（索引 + 可视化）

## 目标

为项目内人物提供出场追踪视图：基于人物卡片名称（可扩展别名）对正文建立位置索引，展示出场章节/片段并可一键跳转到正文证据位置；不输出一致性判断。

## 依赖

- Sprint 5：人物卡片（name/aliases）持久化与项目隔离
- 文档定位能力（章节级/段落级跳转）

## 实现步骤

1. 名称与别名匹配：
   - 基于人物卡 name + aliases 做字符串/规则匹配
   - 记录每次命中的证据位置与上下文片段
2. 出场索引刷新：
   - 人物卡 name/aliases 变更后可触发刷新
   - 失败可观测且可重试
3. 出场视图：
   - 展示人物列表（次数统计/最近出场）
   - 人物详情展示命中列表与跳转
4. 中立呈现：
   - 对歧义命中仅展示证据，不给“误用/写错”结论

## 验收标准

- [ ] 出场追踪视图可用，且可为人物生成出场记录
- [ ] 每条出场记录可跳转回正文证据位置
- [ ] 人物名/别名修改后，索引可刷新且不会 silent failure
- [ ] 不出现任何自动一致性/矛盾提示
