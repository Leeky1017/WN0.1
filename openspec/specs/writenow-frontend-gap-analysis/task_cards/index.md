# Task Cards：writenow-frontend 差距补全

任务卡片按优先级分组：

- **P0**：后端已有但前端无入口（最高 ROI，优先补齐入口 + 最小可用交互）
- **P1**：闭环完善（自定义技能/对话管理/记忆学习/Judge 管理/编辑器手感）
- **P2**：专业写作能力（索引维护/Corkboard/写作目标）
- **P3**：完整产品能力（时间线/伏笔追踪）

## 任务卡片规范（最小必含）

每张任务卡必须包含：

- Goal（目标）
- Dependencies（依赖）
- Expected File Changes（预期文件变更）
- Acceptance Criteria（验收标准）
- Tests（测试要求：至少 1 条真实 E2E）
